## B6: Math Problem-Solving Flow Adventures
**Duration:** 10-11 minutes  
**Learning Objective:** Students will solve age-appropriate problems involving flow rates, distribution networks, and resource allocation.

### Grade 3 Flow Network Problem Adventures
**Problem Set 1: Flow Rate and Distribution**

**Story Problem A: River Flow Calculation**
"A river brings 120 gallons of water per hour to a distribution point. Community A needs 40 gallons per hour, Community B needs 35 gallons per hour, and Community C needs 30 gallons per hour. Will there be enough water for all communities? How much water will be left over?"
- Calculate: 40 + 35 + 30 = 105 gallons needed; 120 - 105 = 15 gallons left over per hour!

**Story Problem B: Network Path Planning**
"A delivery network needs to connect 1 main warehouse to 6 different stores. If a direct connection costs $10 per store, but a branching network costs $15 for the main branch plus $5 for each store branch, which system costs less?"
- Calculate: Direct = 6 × $10 = $60; Branching = $15 + (6 × $5) = $45. Branching network saves $15!

**Story Problem C: Distribution Time**
"Water takes 2 hours to flow through the main channel, plus 1 additional hour through each branch channel. If there are 3 branch channels, what's the total time for water to reach the farthest point?"
- Calculate: 2 hours (main) + 1 hour (branch) = 3 hours total flow time!

### Problem Set 2: Real-Life Applications
**Challenge Adventures:**

**Design Problem:** "Design a supply distribution network for 12 classrooms arranged in 3 groups of 4. If the main supply room can serve 3 secondary distribution points, and each secondary point serves 4 classrooms, how many total supply pathways are needed?"
- Calculate: 1 main to 3 secondary = 3 pathways; 3 secondary × 4 classrooms each = 12 pathways; Total = 15 pathways!

**Efficiency Problem:** "Compare two distribution systems. System A uses 1 main pathway plus 8 branches (9 total pathways). System B uses 3 separate pathways to serve the same 8 endpoints. Which system uses fewer total pathways and might be more efficient?"
- Compare: System A = 9 pathways; System B = 3 pathways. System B is simpler but System A provides better redundancy!

### Problem-Solving Helper Steps
**How to Solve Flow Network Problems:**
1. **Identify the flow:** What needs to be distributed and where does it need to go?
2. **Calculate capacity:** How much total flow or distribution capacity is needed?
3. **Design pathways:** What's the most efficient way to connect sources to destinations?
4. **Check adequacy:** Does the system provide enough capacity for all needs?
5. **Consider reliability:** What happens if part of the system stops working?

### Real-Life Math Applications
**Where These Skills Help:**
- Planning efficient routes for getting from home to school to activities
- Understanding why utility bills (water, electricity) work the way they do
- Designing fair sharing systems for family or classroom resources
- Recognizing efficient vs. wasteful distribution in community systems
