## B6: Mathematical Problem-Solving Workshop
**Duration:** 10-12 minutes  
**Learning Objective:** Students will solve multi-step problems involving tool efficiency, leverage calculations, and optimization analysis.

### Grade 4 Tool Design Problems
**Problem Set 1: Geometry and Measurement**

**Problem A: Optimal Spade Angle**
"A spade blade makes a 20° angle with the handle. If the handle is 3 feet long and the blade is 8 inches wide, what is the total cutting edge length when the spade is viewed from the side?"
- Solution: Use basic trigonometry concepts appropriate for Grade 4

**Problem B: Leverage Calculation**
"Maya uses a spade with a 4-foot handle to lift dirt. If she applies 10 pounds of force at the handle end, and the fulcrum (ground contact) is 6 inches from the blade, how much lifting force does the blade provide?"
- Solution: Leverage ratio = handle length ÷ fulcrum distance; Force multiplication calculation

**Problem C: Work Efficiency**
"Two students dig the same size hole. Student A uses an optimized spade and completes the job in 15 minutes. Student B uses a less efficient tool and takes 25 minutes. What percentage more efficient is the optimized tool?"
- Solution: Efficiency comparison calculation; 25-15=10; 10÷25=40% more efficient

### Problem Set 2: Design and Optimization
**Challenge Problems:**

**Material Removal Problem:** "A construction project needs to remove 100 cubic feet of soil. Calculate how many spadefuls this requires if each spadeful removes 0.25 cubic feet. How would this change with a larger, more efficient spade that removes 0.4 cubic feet per scoop?"

**Tool Selection Problem:** "Compare three different cutting tools for a specific job: Tool A completes the task in 8 minutes but costs $15; Tool B takes 12 minutes but costs $8; Tool C takes 6 minutes but costs $25. Which tool provides the best value considering both time and money?"

### Problem-Solving Strategies
**Steps for Tool Optimization Problems:**
1. **Define the task clearly:** What exactly needs to be accomplished?
2. **Identify key variables:** Time, force, cost, precision, safety requirements
3. **Calculate performance metrics:** Efficiency, cost-effectiveness, speed, accuracy
4. **Compare alternatives systematically:** Use consistent measures across options
5. **Consider trade-offs:** Balance multiple factors to find optimal solution
