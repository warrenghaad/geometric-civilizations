# eTextbook Working Area

Use this folder to assemble eTextbook chapters from section drafts.

Flow:
- Create a Week table to map Day A/B sections (Week thesis + Day rows).
- Draft sections (Day A/B) live here as working copies.
- Aggregate sections into an eTextbook chapter (working copy) here.
- The master template stays in `02_DATABASE/Templates and SSOTs/eTextbook_Chapter_Master.md`.

Suggested files:
- `Week_table_W1.md` — weekly table for mapping sections.
- `DayA_W1.md`, `DayB_W1.md` — section drafts for Week 1 (use Day A/B sections).
- `eTextbook_W1.md` — assembled chapter for Week 1 (based on master template).
