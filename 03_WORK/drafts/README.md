# Drafts (Research Notes)

Use this folder for Research Notes that summarize sources and guide lesson section drafting.

Each Research Note should include:
- Summary
- TOC
- Highlights: important vs. irrelevant
- Annotations/recommendations (where to place content/ideas)
- Guiding prompt (fill as needed):
  "Metaphor of [ ] narrated in [ ] visualized by [geo elem] found in [myth art] and in [rituals] needing [material objects with shape + design]. How to draw [myth art]. Some of [myth+geo elem] appear in architecture/civic projects like [bridge]. Myth narrates visual semiotics metaphorically (Day A review) and functionally. [geo element] has [math properties]. When [transformation] → [scientific event]. Optional: combined with [prior discovery] → [process]. Review math/problem set; project."

Workflow:
1) Drop raw files in `01_INPUT/unprocessed/`; auto-tag into `01_INPUT/processed/`.
2) Run the summarizer (auto_summarize.py) to generate a Research Note here.
3) Use the Research Note to fill DayA/DayB section drafts in `03_WORK/etextbook_sections/` and map them in the Week table.
4) Assemble the eTextbook chapter in `03_WORK/etextbook_sections/eTextbook_W1.md` (copy of master template); master stays in `02_DATABASE`.
5) Export final to `04_OUTPUT/html_exports/`.
