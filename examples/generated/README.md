# Generated Content Examples

This directory contains example outputs generated by the curriculum bots.

## Files

### circles-lesson.json
Example lesson about circles in Ancient Egypt generated by the Content Generator Bot.

**Generated with:**
```bash
npm run bot:content -- --civilization ancient-egypt --topic circles
```

### circles-quiz.json
Example quiz about circles generated by the Quiz Generator Bot.

**Generated with:**
```bash
npm run bot:quiz -- --topic circles --civilization ancient-egypt --questions 5
```

### hexagon-visualization.html
Interactive hexagon visualization with Islamic Golden Age styling generated by the Visualization Generator Bot.

**Generated with:**
```bash
npm run bot:visualize -- --shape hexagon --civilization islamic-golden-age
```

**To view:** Open the HTML file in a web browser.

## Purpose

These examples demonstrate:
- What the bots can generate
- The structure and format of outputs
- Quality and detail level
- How different civilizations are represented

## Using These Examples

1. **As Templates**: Use as starting points for your own content
2. **For Learning**: Study the structure to understand curriculum design
3. **For Testing**: Use to test the assessment bot
4. **For Inspiration**: See what's possible with the system

## Generating Your Own

Try these commands to generate similar content:

```bash
# Different civilization
npm run bot:content -- --civilization ancient-greece --topic triangles

# Different difficulty
npm run bot:quiz -- --topic pythagorean-theorem --difficulty beginner --questions 10

# Different shape
npm run bot:visualize -- --shape star --civilization maya
```

## Note

These are AI-assisted outputs and should be reviewed and customized before use with students.
