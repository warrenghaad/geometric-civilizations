- **Padding:** 60px all sides
- **Text Color:** White (#FFFFFF)
- **Font:** Roboto Regular
- **Size:** 48pt (body), 32pt (footnote)
- **Line Height:** 1.5
- **Alignment:** Centered
- **Drop Shadow:** 2px blur, 40% opacity

#### Footnote Numbers
- **Font:** Roboto Regular
- **Size:** 28pt
- **Color:** Sky Gold (#FFD24A)
- **Position:** Bottom right corner, 40px margin
- **Style:** Superscript

For each page transition:
- **Duration:** 12-18 seconds per page
- **Entry:** Fade in (0.5s)
- **Exit:** Fade out (0.5s)

- Remove all animations
- Static transitions only

## Quick Start Guide

### Creating Template 1 (Myth) in Canva

1. **Create New Design**
   - Choose "Presentation (16:9)"
   - Set to 1920×1080 px

2. **Design Page 1**
   - Upload `meso_G1_1A-IMG-ILL_myth_01_sun_river.png`
   - Set as full-bleed background
   - Add rectangle at bottom 25% (gradient black 0→50%)
   - Add text box: Roboto 48pt, white, centered
   - Add footnote: Roboto 28pt, Sky Gold, bottom right

3. **Duplicate for Pages 2-14**
   - Use "Duplicate page" feature
   - Replace background image on each
   - Update text for each sentence
   - Update footnote number

4. **Save as Template**
   - File → "Save as template"
   - Name: "MESO Myth Sequence Template"
   - Share with team

### Creating Template 2 (Presentation) in Canva

1. **Create Master Layouts**
   - Make one example of each layout type (1-6)
   - Save each as a separate page
   - Lock background elements

2. **Build Full Deck**
   - Duplicate appropriate layout for each slide
   - Insert content per slide spec above
   - Add speaker notes (if Canva supports)

3. **Add Animations (Optional)**
   - Select element → Animate
   - Choose "Fade" or "Slide"
   - Set duration to 0.5-1s

4. **Export**
   - Download → PowerPoint
   - Also download → PDF (for backup)
