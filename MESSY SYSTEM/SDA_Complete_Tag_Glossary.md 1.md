# SDA Complete Tag Glossary
File: 02_SDA_Complete_Tag_Glossary.md

Usage
- Day A: 3–5 tags from Semiotics/Embodied/Material + ≤2 architecture.
- Bridge: 2–4 architecture tags.
- Day B: 4–5 from Math/Design/Institution/Discovery + 1–2 architecture.
- Always include a #g/* element tag.

Crosswalk to Image Ontology
- dimensions → #g/*
- dimensionMapping → #arch/grammar/module, #standard/module
- geometricElements → #g/*, #semiotics/grammar/pattern
- operations → #math/tiling/plane, #math/graph, #arch/technique/*
- materials → #material, #design/constraint
- techniques → #everyday/tool, #arch/technique/*
- culturalContexts → #arch/institution/branding, #market/exchange
- mathConcepts → #math/*
- mythologyRefs / mythology_symbols → #semiotics/*

A) Semiotics and Myth (Day A)
- #semiotics/symbol — Learned cultural sign. [mythology_symbols]
- #semiotics/icon — Resemblance (sun disk, star). [mythology_symbols]
- #semiotics/metaphor — Mythic mapping (“light as justice,” “cycle as time”).
- #semiotics/grammar/pattern — Motif rules, repetition, interlock.
- #semiotics/cosmogram — Ordered map or plan of the cosmos.

B) Embodied Practice (Day A)
- #embodied/gesture — Ritualized movement tied to geometry.
- #embodied/rhythm — Counts/steps; processional beats.
- #embodied/sensorimotor/rotation — Turning, facing, reorienting.
- #embodied/proprioception/centering — Balance, centering practices.

C) Material Culture (Day A light)
- #artifact — Portable objects, tablets, seals.
- #material — Stone, clay, glaze, textile.
- #ritual/space — Altars, thresholds, sanctuaries.
- #everyday/tool — Cords, compasses, looms.

D) Architecture Bridge (both days)
- #arch/form-function — Visible form serves a structural or civic function.
- #arch/grammar/module — Bays, bricks, unit grids.
- #arch/standard/measure — Rods, course heights, canonical sizes.
- #arch/flow/procession — Processional wayfinding; circulation.
- #arch/technique/… — Corbel, vault, pier, buttress, glazing.
- #arch/institution/branding — State/cult identity expressed in building.

E) Math and Structure (Day B)
- #magic/m — Math driver foregrounded.
- #math/ratio — Quantitative relationships, e.g., 3:2, 1:√2.
- #math/symmetry/rotational — Orders, centers, axes.
- #math/tiling/plane — Gapless coverings, tessellations.
- #math/graph — Adjacency, routing, networks.

F) Design and Standards (Day B)
- #design/specification — Explicit design requirements.
- #design/constraint — Limits: span, load, climate, tools.
- #standard/module — Reusable units enabling scaling.
- #standard/measure — Measurement systems and tools.
- #standard/protocol — Workshop steps, QA, labor staging.

G) Institutions and Logistics (Day B)
- #magic/c — Comptroller/patronage/administration.
- #institution/labor — Crews, guilds, corvée.
- #institution/workshop — Production settings.
- #infrastructure/transport — Roads, canals, gates.
- #market/exchange — Storage, pricing, fair weights.

H) Discovery → Innovation → Invention (Day B)
- #discovery — New relation or regularity recognized.
- #innovation — Improved method applying discovery.
- #invention — New device/system capability.

I) Geometric Elements (both days)
- #g/circle, #g/radial, #g/star, #g/triangle, #g/square, #g/zigzag, #g/spiral, #g/crescent, #g/grid, #g/composite
